## 运行环境
- 操作系统：Windows
- Python版本:3.7以上 
- 依赖参照requirements.txt

> pip install -r requirements.txt

## 编译方法
- 安装完依赖后直接运行图片华容道中的main.py
- main.py 整个游戏项目
- test.py 用于获取接口数据进行测试
- AI大比拼.py 为AI大比拼
> python main.py
 
> python test.py

> python AI大比拼.py

## 使用方法 
- 可以直接执行run文件夹中的main.exe文件
> main.exe
